Q: What is flexbox? Where to use it and where not to use it?

Flexbox is a layout model that allows to align and distribute space within a container, it's a great tool to use for responsive design. Flexbox layout is most appropriate to the components of an application and small scale-layout, while the Grid layout is intended for larger scale layouts.

Q: How to avoid Callback Hell in JavaScript?

Write comments for everyone to understand what is going on. Split functions into smaller functions. Use of promises, Async/await can also help.

Q: What does the following CSS code snippet mean? #log>span
